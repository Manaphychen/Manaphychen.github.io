### 单链表的应用实例
使用带 head 头的单向链表实现人物的增删改查

1. 定义节点
```java
public class Node {
    public int id;
    public String name;
    public Node next; // 指向下一个节点

    public Node(int id, String name) {
        this.id = id;
        this.name = name;
    }

    @Override
    public String toString() {
        return "Node{" +
                "id=" + id +
                ", name='" + name + '\'' +
                '}';
    }
}
```

2. 定义 SingleLinkedList 管理节点
```java
public class SingleLinkedList {
    // 先初始化一个头节点, 头节点不要动, 不存放具体的数据
    private Node head = new Node(0, "");

    // 返回头节点
    public Node getHead() {
        return head;
    }

    // 添加节点到单向链表
    public void add(Node node) {
        // 因为 head 节点不能动，因此我们需要一个辅助遍历 temp
        Node temp = head;
        // 遍历链表,找到最后

        while (temp.next != null) {// 已经到链表的最后
            // 如果没有找到最后, 将temp后移
            temp = temp.next;
        }
        // 当退出while循环时，temp就指向了链表的最后 将最后这个节点的 next 指向新的节点
        temp.next = node;
    }

    // 添加节点时,将根据 id 排序.如果 id 相同,则替换
    public void addByOrder(Node node) {
        // 因为头节点不能动，因此我们仍然通过一个辅助指针(变量)来帮助找到添加的位置
        Node temp = head;
        while (true) {
            if (temp.next == null) { // 说明temp已经在链表的最后
                break;
            }
            if (temp.next.id > node.id) {
                break;
            }
            if (temp.next.id == node.id) { // 说明希望添加的节点的 id 已然存在
                // 将新的节点的内容替换掉原来的内容
                temp.next.name = node.name;
                return;
            }
            // 后移，遍历当前链表
            temp = temp.next;
        }
        // 插入到链表中, temp的后面
        node.next = temp.next;
        temp.next = node;

    }

    // 查找
    public void get(int id) {
        //判断是否空
        if (head.next == null) {
            System.out.println("链表为空~");
            return;
        }
        // 找到需要修改的节点, 定义一个辅助变量
        Node next = head.next;
        while (true) {
            if (next.id == id) {
                System.out.println(next);
                break;
            }
            if (next.next == null) { // 已经遍历完链表
                return;
            }
            next = next.next;
        }
    }

    // 修改链表的节点信息,根据 id 来修改 id不变
    public void update(Node node) {
        //判断是否空
        if (head.next == null) {
            System.out.println("链表为空~");
            return;
        }
        // 找到需要修改的节点, 定义一个辅助变量
        Node next = head.next;
        while (true) {
            if (next.id == node.id) {
                next.name = node.name;
                return;
            }
            if (next.next == null) { // 已经遍历完链表
                return;
            }
            next = next.next;
        }
    }

    // 删除节点
    public void delete(int id) {
        Node temp = head;
        while (temp.next != null) {
            if (temp.next.id == id) {
                temp.next = temp.next.next;
                return;
            }
            temp = temp.next;
        }
    }

    public void list() {
        //判断链表是否为空
        if (head.next == null) {
            System.out.println("链表为空");
            return;
        }
        Node temp = head.next;
        while (temp != null) {
            System.out.println(temp);
            temp = temp.next;
        }
    }
}
```

```java
public int length() {
    int length = 0;
    if (head.next == null) {
        return length;
    }

    Node node = head.next;
    while (node != null) {
        length++;
        node = node.next;
    }
    return length;
}
```

2. 查找单链表中的倒数第k个结点
```java
public Node findLastIndexNode(int index) {
    if (head.next == null) {
     return null;
    }
    int length = length();
    // 先做校验
    if (index <= 0 || index > length) {
        return null;
    }
    Node node = head.next;
    for (int i = 0; i < length - index; i++) {
        node = node.next;
    }
    return node;
}
```

3. 单链表的反转
```java
public void reverseList() {
    Node node = head.next;
    // 如果当前链表为空，或者只有一个节点，无需反转，直接返回
    if (node == null || node.next == null) {
        return;
    }
    Node reverseHead = new Node(0, "");
    Node temp;
    // 遍历原来的链表，每遍历一个节点，就将其取出，并放在新的链表 reverseHead 的最前端
    while (node != null) {
        // 先暂时保存当前节点的下一个节点，因为后面需要使用
        temp = node.next;
        // 将 node 的下一个节点指向新的链表的最前端
        node.next = reverseHead.next;
        // 将 node 连接到新的链表上
        reverseHead.next = node;
        // 让 node 后移
        node = temp;
    }
    // 将 head.next 指向 reverseHead.next,实现单链表的反转
    head.next = reverseHead.next;
}
```

### 双向链表应用实例
管理单向链表的缺点分析

1. 单向链表，查找的方向只能是一个方向，而双向链表可以向前或者向后查找。
2. 单向链表不能自我删除，需要靠辅助节点 ，而双向链表，则可以自我删除，所以前面我们单链表删除时节点，总是找到 temp,temp 是待删除节点的前一个节点

使用带 head 头的双向链表实现人物的增删改查

1. 定义节点
```java
public class DoubleNode {
    public int id;
    public String name;
    public DoubleNode next; // 指向下一个节点
    public DoubleNode pre;

    public DoubleNode(int id, String name) {
        this.id = id;
        this.name = name;
    }

    @Override
    public String toString() {
        return "Node{" +
                "id=" + id +
                ", name='" + name + '\'' +
                '}';
    }
}
```

2. 定义 DoubleLinkedList 管理节点
```java
public class DoubleLinkedList {
    // 先初始化一个头节点, 头节点不要动, 不存放具体的数据
    private final DoubleNode head = new DoubleNode(0, "");

    public DoubleNode getHead() {
        return head;
    }

    public void list() {
        if (head.next == null) {
            System.out.println("链表为空");
            return;
        }
        DoubleNode temp = head.next;
        while (temp != null) {
            System.out.println(temp);
            temp = temp.next;
        }
    }

    // 添加节点到双向链表
    public void add(DoubleNode node) {
        DoubleNode temp = head;
        while (temp.next != null) {
            temp = temp.next;
        }
        // 将最后这个节点的 next 指向新的节点 将新的节点的 pre 指向前一个节点
        temp.next = node;
        node.pre = temp;
    }
    
    // 添加节点时,将根据 id 排序.如果 id 相同,则替换
    public void addByOrder(DoubleNode node) {
        // 因为头节点不能动，因此我们仍然通过一个辅助指针(变量)来帮助找到添加的位置
        DoubleNode temp = head;
        while (true) {
            if (temp.next == null) { // 说明temp已经在链表的最后
                break;
            }
            if (temp.next.id > node.id) {
                break;
            }
            if (temp.next.id == node.id) { // 说明希望添加的节点的 id 已然存在
                // 将新的节点的内容替换掉原来的内容
                temp.next.name = node.name;
                return;
            }
            // 后移，遍历当前链表
            temp = temp.next;
        }
        // 插入到链表中, temp的后面
        node.next = temp.next;
        node.pre = temp;
        temp.next = node;
    }

    // 修改链表的节点信息 跟单向链表类似
    public void update(DoubleNode node) {
        if (head.next == null) {
            System.out.println("链表为空~");
            return;
        }
        DoubleNode next = head.next;
        while (true) {
            if (next.id == node.id) {
                next.name = node.name;
                return;
            }
            if (next.next == null) {
                return;
            }
            next = next.next;
        }
    }

    // 从双向链表中删除一个节点
    public void delete(int id) {
        DoubleNode temp = head.next;
        while (temp != null) {
            if (temp.id == id) {
                // 将该节点的上一个节点的下一个节点指向该节点的下一个节点
                temp.pre.next = temp.next;
                // 如果是最后一个节点,就不需要执行下面这句话,否则出现NPE
                if (temp.next != null) {
                    // 将该节点的下一个节点的上一个节点指向该节点的上一个节点
                    temp.next.pre = temp.pre;
                }
                return;
            }
            temp = temp.next;
        }
    }

}
```

### 单向环形链表应用场景
**Josephu(约瑟夫、约瑟夫环) 问题**<br />Josephu 问题为：设编号为 1，2，… n 的 n 个人围坐一圈，约定编号为 k（1<=k<=n）的人从 1 开始报数，数到 m 的那个人出列，它的下一位又从 1 开始报数，数到 m 的那个人又出列，依次类推，直到所有人出列为止，由此产生一个出队编号的序列。

提示：用一个不带头结点的循环链表来处理 Josephu  问题：先构成一个有 n 个结点的单循环链表，然后由 k 结点起从 1 开始计数，计到 m 时，对应结点从链表中删除，然后再从被删除结点的下一个结点又从 1 开始计数，直到最后一个结点从链表中删除算法结束。<br />![](https://cdn-us.imgs.moe/2024/03/09/1612022273585-2f686bd7-0e24-40a1-8bee-b5681372d560_ZbLpciYWCu.png)<br />**Josephu 问题的代码实现**

2. 定义 CircleSingleLinkedList 管理节点
```java
public class CircleSingleLinkedList {
    // 创建一个 first 节点,当前没有 id
    private Node first = null;

    /**
     * 构建从id从 1 到 nums 的环形链表
     *
     * @param nums 构建的个数
     */
    public void add(int nums) {
        if (nums < 1) {
            System.out.println("nums不得小于1");
            return;
        }
        // 创建辅助指针,帮助构建环形链表
        Node currentNode = null;
        // 使用 for 循环创建环形链表
        for (int i = 1; i <= nums; i++) {
            Node node = new Node(i);
            // 如果是第一个节点
            if (i == 1) {
                first = node;
                // 构成环
                first.next = first;
                // 让辅助指针指向第一个节点
                currentNode = first;
            } else {
                currentNode.next = node;
                node.next = first;
                currentNode = node;
            }
        }
    }

    // 遍历环形链表
    public void list() {
        // 判断链表是否为空
        if (first == null) {
            System.out.println("链表为空");
            return;
        }
        // 因为 first 不能动,所以需要辅助指针完成遍历
        Node currentNode = first;
        while (true) {
            System.out.printf("id: %d \n", currentNode.id);
            // 如果当前节点的下一个节点是 first 节点 就终止遍历
            if (currentNode.next == first) {
                break;
            }
            currentNode = currentNode.next;
        }
    }

    /**
     * 约瑟夫问题
     *
     * @param startId  表示从第几个id开始数
     * @param countNum 表示数几下
     * @param nums     表示最初有几个在圈中
     */
    public void josepfuQuestion(int startId, int countNum, int nums) {
        if (first == null || startId < 1 || startId > nums) {
            System.out.println("参数有误!");
            return;
        }
        // 创建辅助指针
        Node helper = first;
        // 让 helper 指向最后一个节点
        while (helper.next != first) {
            helper = helper.next;
        }
        // 小孩报数前，先让 first 和	helper 移动 k - 1 次
        for (int i = 0; i < startId - 1; i++) {
            first = first.next;
            helper = helper.next;
        }
        // 当小孩报数时，让 first 和 helper 指针同时移动 m-1 次, 然后出圈
        // 当 helper == first 时,循环结束
        while (helper != first) {
            // 让 first 和 helper 指针同时移动 countNum - 1 次
            for (int i = 0; i < countNum - 1; i++) {
                first = first.next;
                helper = helper.next;
            }
            System.out.println("id出圈:" + first.id);
            first = first.next;
            helper.next = first;
        }
        System.out.println("最后留在圈中的id为:" + first.id);
    }

}
```
