### 8.4 切片

切片是指对操作的对象截取其中一部分的操作。字符串、列表、元组都支持切片操作。 

切片的语法：[起始:结束:步长]，也可以简化使用 [起始:结束] 

注意：选取的区间从"起始"位开始，到"结束"位的前一位结束（不包含结束位本身)，步长表示选取间隔。

```python
s = 'hello world'

print(s[0:4])  # hell
print(s[4:])  # o world
print(s[:4])  # hell
print(s[1:8:2])  # el o
print(s[:])  # hello world
print(s[-4:-1])  # orl
# 字符串的逆序输出
print(s[::-1])  # dlrow olleh
# 隔一个字符输出
print(s[::2])  # hlowrd

```

## 11. 引用

```python
# 可以使用 id() 查看变量的引用, 可以将id 值认为是内存地址的别名
# python中数据值的传递的是引用
# 赋值运算符可以改变变量的引用

# 将数据10 存储到变量 a 中,  本质是将数据10 所在内存的引用地址保存到变量 a 中
a = 10
```

### 可变与不可变类型

```python
# 类型的可变与不可变: 在不改变变量引用的前提下,能否改变变量中引用中的数据,
# 如果能改变是可变类型, 如果不能改变,是不可变类型
# int float bool str list tuple dict
# 不可变类型:  int float bool str  tuple
# 可变类型: list dict

a = 1000
b = 1000
print(id(a) == id(b))  # True  python中的内存优化,对于不可变类型进行的

a = 'hello'
b = 'hello'
print(id(a) == id(b))  # True python中的内存优化,对于不可变类型进行的,

my_tuple = (1, 2)
my_tuple1 = (1, 2)
print(id(my_tuple) == id(my_tuple1))  # True

my_list = [1, 2, 3]
my_list1 = [1, 2, 3]
print(id(my_list) == id(my_list1))  # False
```

```python
    :param func: 函数,要进行的运算
    :return: 运算的结果
    """
    print('其他的函数代码...')
    num = func(a, b)
    print(num)


def add(a, b):
    return a + b


# 调用
```

### 12.2 文件的读写

```python
# 模式 a 追加
file = open('test.txt', 'w')
# 写数据
file.write('hello world\n' * 2)
file.writelines("abc")
file.close()

# 读数据
fp = open('test.txt', 'r')
# 默认情况下 read是一字节一字节的读 效率比较低
fp.read(5)  # 最多读取5个数据
fp.read()  # 从上次读取的位置继续读取剩下的所有数据

fp = open('test.txt')  # 访问模式默认为 r
# 读取一行
fp.readline()
# 一行一行读,读取所有数据后放入列表
print(fp.readlines())  # ['hello world\n', 'abc']
fp.close()
```

### 12.3 序列化与反序列化

通过文件操作，我们可以将字符串写入到一个本地文件。但是，如果是一个对象(例如列表、字典、元组等)，就无 法直接写入到一个文件里，需要对这个对象进行序列化，然后才能写入到文件里。

设计一套协议，按照某种规则，把内存中的数据转换为字节序列，保存到文件，这就是序列化，反之，从文件的字 节序列恢复到内存中，就是反序列化。

Python中提供了JSON这个模块用来实现数据的序列化和反序列化。

```python
import json

file = open('name.txt', 'w')

nameList = ['zs', 'ls', 'ww', 'cl']

# 序列化
# 将python对象 变成 json 字符串
names = json.dumps(nameList)
# 写入
file.write(names)

# 以上转换和写入可以使用以下方法替代
file = open('name.txt', 'w')
json.dump(nameList, file)
file.close()

# 反序列化
# 将 json 字符串 变成 python 对象
fp = open('name.txt')
content = fp.read()
print(type(content))  # <class 'str'>
result = json.loads(content)
print(type(result))  # <class 'list'>

# 以上读取和转换可以使用以下方法替代
fp = open('name.txt')
result = json.load(fp)
print(type(result))  # <class 'list'>

fp.close()
```
