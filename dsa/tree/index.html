数据结构与算法 - 树结构基础

# 第九章 树结构基础部分

# 9.1 二叉树

# 9.1.1 为什么需要树这种数据结构

  1. 数组存储方式的分析
    • 优点：通过下标方式访问元素，速度快。对于有序数组，还可使用二分查找提高检索速度。
    • 缺点：如果要检索具体某个值，或者插入值(按一定顺序)会整体移动，效率较低
  2. 链式存储方式的分析
    • 优点：在一定程度上对数组存储方式有优化(比如：插入一个数值节点，只需要将插入节点，链接到链表中即可， 删除效率也很好)。
    • 缺点：在进行检索时，效率仍然较低，比如(检索某个值，需要从头节点开始遍历)
  3. 树存储方式的分析
    能提高数据存储，读取的效率,  比如利用 二叉排序树(Binary Sort Tree)，既可以保证数据的检索速度，同时也可以保证数据的插入，删除，修改的速度。

# 9.1.2 树示意图

  • 节点: 树上一个存储数据的单元
  • 根节点: 最顶层的节点
  • 父节点: 节点上连接的上一层节点
  • 子节点: 节点向下连接的节点
  • 叶子节点: 最后一层的节点 即没有子节点的节点
  • 节点权: 节点值
  • 路径: 从根节点出发到节点所经过的路线
  • 层: 节点在从根节点开始往下数 在第几层
  • 高度: 整棵树一共有多少层
  • 子树: 一个节点即他的子节点组成一个子树
  • 森林: 多棵子树组成森林

# 9.1.3 二叉树的概念

  1. 树有很多种，每个节点最多只能有两个子节点的一种形式称为二叉树。
  2. 二叉树的子节点分为左节点和右节点。

  1. 如果该二叉树的所有叶子节点都在最后一层，并且节点总数= 2^n -1 , n 为层数，则我们称为满二叉树。
  2. 如果该二叉树的所有叶子节点都在最后一层或者倒数第二层，而且最后一层的叶子节点在左边连续，倒数第二层的叶子节点在右边连续，我们称为完全二叉树。

# 9.1.4 二叉树遍历的说明

使用前序，中序和后序对下面的二叉树进行遍历


前序遍历: 先输出父节点，再遍历左子树和右子树 [1 2 4 5 3 6 7]
中序遍历: 先遍历左子树，再输出父节点，再遍历右子树 [4 2 5 1 6 3 7]
后序遍历: 先遍历左子树，再遍历右子树，最后输出父节点 [4 5 2 6 7 3 1]
小结: 看输出父节点的顺序，就确定是前序，中序还是后序

# 9.1.5 二叉树遍历应用实例

代码实现
定义节点

public class Node<T> {
    T element;
    Node<T> left;
    Node<T> right;

    public Node(T element) {
        this.element = element;
    }

    @Override
    public String toString() {
        return element.toString();
    }

    /**
     * 前序遍历
     */
    public void preorder() {
        // 先输出父节点
        System.out.print(this + " ");

        // 递归向左子树前序遍历
        if (this.left != null) {
            this.left.preorder();
        }

        // 递归向左子树前序遍历
        if (this.right != null) {
            this.right.preorder();
        }
    }

    /**
     * 中序遍历
     */
    public void inorder() {
        // 先向左递归子树中序遍历
        if (this.left != null) {
            this.left.inorder();
        }

        // 输出父节点
        System.out.print(this + " ");

        // 递归向右子树中序遍历
        if (this.right != null) {
            this.right.inorder();
        }
    }

    /**
     * 后序遍历
     */
    public void postorder() {
        // 先向左递归子树后序遍历
        if (this.left != null) {
            this.left.postorder();
        }
        // 再递归向右子树后序遍历
        if (this.right != null) {
            this.right.postorder();
        }
        // 最后输出父节点
        System.out.print(this + " ");
    }

}

public class BinaryTree<T> {

    public static void main(String[] args) {
        // 创建一颗二叉树
        BinaryTree<Integer> tree = new BinaryTree<>();
        // 创建节点
        Node<Integer> root = new Node<>(1);
        Node<Integer> node2 = new Node<>(2);
        Node<Integer> node3 = new Node<>(3);
        Node<Integer> node4 = new Node<>(4);
        Node<Integer> node5 = new Node<>(5);

        // 手动创建该二叉树
        root.left = node2;
        root.right = node3;
        node3.right = node4;
        node3.left = node5;
        tree.root = root;


        // 测试前中后序遍历
        tree.traverse(TraverseType.PREORDER); //1 2 3 5 4
        tree.traverse(TraverseType.INORDER); //2 1 5 3 4
        tree.traverse(TraverseType.POSTORDER); //2 5 4 3 1
    }

    Node<T> root;

    /**
     * 遍历
     *
     * @param type 遍历类型
     */
    public void traverse(TraverseType type) {
        if (this.root == null) {
            System.out.println("二叉树为空");
            return;
        }
        switch (type) {
            case PREORDER:
                System.out.print("前序遍历：\t");
                root.preorder();
                System.out.println();
                break;
            case INORDER:
                System.out.print("中序遍历：\t");
                root.inorder();
                System.out.println();
                break;
            case POSTORDER:
                System.out.print("后序遍历：\t");
                root.postorder();
                System.out.println();
                break;
        }
    }
}

# 9.1.6 二叉树查找指定节点

查找也有三种查找方式 前序 中序 后序遍历查找

Node添加查找方法

/**
 * 前序遍历查找
 *
 * @param element 根据 element 查找
 * @return 返回节点, 如果没找到返回 null
 */
public Node<T> preorderSearch(T element) {
    System.out.println("前序查找");
    // 1.先比较当前节点是不是
    if (this.element == element) {
        // 是的话直接返回当前节点
        return this;
    }
    Node<T> result = null;
    // 2.判断当前节点的左子节点是否为空,如果不为空,则递归前序查找,找到就返回
    if (this.left != null) {
        result = this.left.preorderSearch(element);
    }
    // 如果 result 不为空,说明在左子树找到节点
    if (result != null) {
        return result;
    }
    // 3.判断当前的节点的右子节点是否为空，如果不空，则继续向右递归前序查找
    if (this.right != null) {
        result = this.right.preorderSearch(element);
    }
    // 不论是否为空 都直接返回
    return result;
}

/**
 * 中序遍历查找
 */
public Node<T> inorderSearch(T element) {
    Node<T> result = null;
    // 1.判断当前节点的左子节点是否为空,如果不为空,则递归中序查找,找到就返回
    if (this.left != null) {
        result = this.left.inorderSearch(element);
    }
    if (result != null) {
        return result;
    }
    System.out.println("中序查找");
    // 2.左子节点没找到,就和当前节点比较，如果是则返回当前节点
    if (this.element == element) {
        return this;
    }
    // 3.否则继续进行右递归的中序查找
    if (this.right != null) {
        result = this.right.inorderSearch(element);
    }
    return result;
}

/**
 * 后序遍历查找
 */
public Node<T> postorderSearch(T element) {
    Node<T> result = null;
    // 1.判断当前节点的左子节点是否为空,如果不为空,则递归后序查找,找到就返回
    if (this.left != null) {
        result = this.left.postorderSearch(element);
    }
    if (result != null) {
        return result;
    }
    // 2.左子节点没找到,则向右子树递归进行后序遍历查找
    if (this.right != null) {
        result = this.right.postorderSearch(element);
    }
    if (result != null) {
        return result;
    }
    System.out.println("后序查找");
    // 3.如果左右子树都没有找到，就比较当前节点是不是
    if (this.element == element) {
        return this;
    }
    return null;
}

/**

 *
 * @param type    类型
 * @param element 元素
 * @return 节点
 */
public Node<T> traverseSearch(TraverseType type, T element) {
    if (this.root == null) {
        return null;
    }
    switch (type) {
        case PREORDER:
            return root.preorderSearch(element);
        case INORDER:
            return root.inorderSearch(element);
        case POSTORDER:
            return root.postorderSearch(element);
    }
    return null;
}

# 9.1.7 二叉树删除节点

代码实现
Node 添加删除方法

/**
 * 删除节点
 * 如果该节点只有一个子节点,则让子节点代替该节点
 * 如果该节点有两个子节点,则让左子节点代替该节点
 *
 * @param element 关键
 */
public void deletePlus(T element) {
    // 1.因为我们的二叉树是单向的，所以我们是判断当前节点的子节点是否需要删除节点，而不能去判断当前这个节点是不是需要删除节点
    // 2.如果当前节点的左子节点不为空，并且左子节点就是要删除节点
    if (this.left != null && this.left.element == element) {
        // 判断左子节点的子节点存在情况
        if (this.left.left == null && this.left.right == null) {
            this.left = null;
            return;
        }
        if (this.left.left != null && this.left.right != null) {
            this.left.left.right = this.left.right;
            this.left = this.left.left;
            return;
        }
        if (this.left.left == null) {
            this.left = this.left.right;
            return;
        }
        this.left = this.left.left;
        return;


    }
    // 3.如果当前节点的右子节点不为空，并且右子节点 就是要删除节点，就将 this.right= null ;并且就返回(结束递归删除)
    if (this.right != null && this.right.element == element) {
        // 判断右子节点的子节点存在情况
        if (this.right.left == null && this.right.right == null) {
            this.right = null;
            return;
        }
        if (this.right.left != null && this.right.right != null) {
            this.right.left.right = this.right.right;
            this.right = this.right.left;
            return;
        }
        if (this.right.left == null) {
            this.right = this.right.right;
            return;
        }
        this.right = this.right.left;
        return;
    }
    // 4.如果第 2 和第 3 步没有删除节点，那么我们就需要向左子树进行递归删除
    if (this.left != null) {
        this.left.delete(element);
    }
    // 5.如果第 4 步也没有删除节点，则应当向右子树进行递归删除
    if (this.right != null) {
        this.right.delete(element);
    }
}

/**
 * 删除节点
 */
public void deleteNode(T element) {
    if (root != null) {
        // 如果只有一个 root 节点,  这里立即判断 root 是不是就是要删除节点
        if (root.element == element) {
            if (root.left != null && root.right != null) {
                root.left.right = root.right;
                root = root.left;
            } else if (root.right != null) {
                root = root.right;
            } else {
                root = null;
            }

        } else {
            // 递归删除
            root.delete(element);
        }
    } else {
        System.out.println("空树,不能删除~");
    }
}

# 9.2 顺序存储二叉树

从数据存储来看，数组存储方式和树的存储方式可以相互转换，即数组可以转换成树，树也可以转换成数组 顺序二叉树通常只考虑完全二叉树

应用实例
八大排序算法中的堆排序，就会使用到顺序存储二叉树
代码实现

public class SeqBinaryTree {

    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5, 6, 7};
        SeqBinaryTree tree = new SeqBinaryTree(arr);
        tree.traverse(TraverseType.PREORDER);
        System.out.println();
        tree.traverse(TraverseType.INORDER);
        System.out.println();
        tree.traverse(TraverseType.POSTORDER);
    }

    // 存储数据节点的数组
    private final int[] arr;

    public SeqBinaryTree(int[] arr) {
        this.arr = arr;
    }

    /**
     * 遍历
     *
     * @param type 遍历类型
     */
    public void traverse(TraverseType type) {
        switch (type) {
            case PREORDER:
                preorder(0);
                break;
            case INORDER:
                inorder(0);
                break;
            case POSTORDER:
                postorder(0);
        }
    }

    /**
     * 前序遍历
     */
    private void preorder(int index) {
        if (arr == null || arr.length == 0) {
            return;
        }
        // 输出当前这个元素
        System.out.print(arr[index] + " ");
        // 向左递归遍历
        if ((index * 2 + 1) < arr.length) {
            preorder(index * 2 + 1);
        }
        // 向右递归遍历
        if (2 * (index + 1) < arr.length) {
            preorder(2 * (index + 1));
        }
    }

    /**
     * 中序遍历
     */
    private void inorder(int index) {
        if (arr == null || arr.length == 0) {
            return;
        }
        // 向左递归遍历
        if ((index * 2 + 1) < arr.length) {
            inorder(index * 2 + 1);
        }
        // 输出当前这个元素
        System.out.print(arr[index] + " ");
        // 向右递归遍历
        if (2 * (index + 1) < arr.length) {
            inorder(2 * (index + 1));
        }
    }

    /**
     * 后序遍历
     */
    private void postorder(int index) {
        if (arr == null || arr.length == 0) {
            return;
        }
        // 向右递归遍历
        if (2 * (index + 1) < arr.length) {
            postorder(2 * (index + 1));
        }
        // 向左递归遍历
        if ((index * 2 + 1) < arr.length) {
            postorder(index * 2 + 1);
        }
        // 输出当前这个元素
        System.out.print(arr[index] + " ");
    }
}

# 9.3 线索化二叉树

对于n个节点的二叉树，在二叉链存储结构中有n+1个空链域，利用这些空链域存放在某种遍历次序下该节点的前驱节点和后继节点的指针，这些指针称为线索，加上线索的二叉树称为线索二叉树。 这种加上了线索的二叉链表称为线索链表，相应的二叉树称为线索二叉树(Threaded BinaryTree)。根据线索性质的不同，线索二叉树可分为前序线索二叉树、中序线索二叉树和后序线索二叉树三种。 **注意：**线索链表解决了无法直接找到该节点在某种遍历序列中的前驱和后继节点的问题，解决了二叉链表找左、右孩子困难的问题。

二叉树的遍历本质上是将一个复杂的非线性结构转换为线性结构，使每个节点都有了唯一前驱和后继（第一个节点无前驱，最后一个节点无后继）。对于二叉树的一个节点，查找其左右子女是方便的，其前驱后继只有在遍历中得到。为了容易找到前驱和后继，有两种方法。一是在节点结构中增加向前和向后的指针，这种方法增加了存储开销，不可取；二是利用二叉树的空链指针。

# 优势与不足

优势

  • 利用线索二叉树进行中序遍历时，不必采用堆栈处理，速度较一般二叉树的遍历速度快，且节约存储空间。
  • 任意一个节点都能直接找到它的前驱和后继节点。

不足

  • 节点的插入和删除麻烦，且速度也较慢。
  • 线索子树不能共用。

编写线索二叉树类

public class ThreadedBinaryTree<T> {

    public static void main(String[] args) {
        // 测试中序线索二叉树功能
        Node<String> A = new Node<>("A");
        Node<String> B = new Node<>("B");
        Node<String> C = new Node<>("C");
        Node<String> D = new Node<>("D");
        Node<String> E = new Node<>("E");
        Node<String> F = new Node<>("F");
        // 二叉树,后面要递归创建,这里先手动创建
        A.left = B;
        A.right = C;
        B.right = D;
        C.left = E;
        C.right = F;

        // 后序遍历时需要设置 父节点
        B.parent = A;
        C.parent = A;
        D.parent = B;
        E.parent = C;
        F.parent = C;
        // 创建线索化二叉树
        ThreadedBinaryTree<String> tree = new ThreadedBinaryTree<>();
        tree.root = A;
        // 测试前序线索化
//        tree.threadedNodes(TraverseType.PREORDER);
//        tree.threadedList(TraverseType.PREORDER);

        // 测试后序线索化
//        tree.threadedNodes(TraverseType.INORDER);
//        tree.threadedList(TraverseType.INORDER);

        // 测试后序线索化
        tree.threadedNodes(TraverseType.POSTORDER);
        tree.threadedList(TraverseType.POSTORDER);

    }

    Node<T> root;
    // 为了实现线索化，需要创建要给指向当前节点的前驱节点的指针
    // 在递归进行线索化时，pre 总是保留前一个节点
    Node<T> pre = null;

    /**
     * 线索化二叉树
     *
     * @param type 类型
     */
    public void threadedNodes(TraverseType type) {
        if (root == null) {
            return;
        }
        switch (type) {
            case PREORDER:
                preThreadedNodes(root);
                break;
            case INORDER:
                inThreadedNodes(root);
                break;
            case POSTORDER:
                postThreadedNodes(root);
        }
    }

    /**
     * 对二叉树进行中序线索化的方法
     *
     * @param node 当前需要线索化的节点
     */
    private void inThreadedNodes(Node<T> node) {
        // 如果 node 为空,则不能线索化
        if (node == null) {
            return;
        }
        // 1.先线索化左子树
        inThreadedNodes(node.left);
        // 2.线索化当前节点
        // 处理当前节点的前驱节点
        if (node.left == null) {
            // 让当前节点的左指针指向前驱节点
            node.left = pre;
            // 修改当前节点的左指针的类型,指向前驱节点
            node.leftType = 1;
        }
        // 处理后继节点
        if (pre != null && pre.right == null) {
            // 让前驱节点的右指针指向当前节点
            pre.right = node;
            pre.rightType = 1;
        }
        // 每处理一个节点后，让当前节点是下一个节点的前驱节点
        pre = node;
        // 3.线索化右子树
        inThreadedNodes(node.right);
    }

    /**
     * 对二叉树进行前序线索化的方法
     *
     * @param node 当前需要线索化的节点
     */
    private void preThreadedNodes(Node<T> node) {
        // 如果 node 为空,则不能线索化
        if (node == null) {
            return;
        }
        // 1.先线索化当前节点
        if (node.left == null) {
            // 让当前节点的左指针指向前驱节点
            node.left = pre;
            // 修改当前节点的左指针的类型,指向前驱节点
            node.leftType = 1;
        }
        // 处理后继节点
        if (pre != null && pre.right == null) {
            // 让前驱节点的右指针指向当前节点
            pre.right = node;
            pre.rightType = 1;
        }
        // 每处理一个节点后，让当前节点是下一个节点的前驱节点
        pre = node;
        // 2.再线索化左子树
        if (node.leftType != 1) {
            preThreadedNodes(node.left);
        }
        // 3.最后线索化右子树
        if (node.rightType != 1) {
            preThreadedNodes(node.right);
        }
    }

    /**
     * 对二叉树进行后序线索化的方法
     *
     * @param node 当前需要线索化的节点
     */
    private void postThreadedNodes(Node<T> node) {
        // 如果 node 为空,则不能线索化
        if (node == null) {
            return;
        }
        // 1.先线索化左子树
        postThreadedNodes(node.left);
        // 2.再线索化右子树
        postThreadedNodes(node.right);
        // 3.最后线索化当前节点
        // 处理当前节点的前驱节点
        if (node.left == null) {
            // 让当前节点的左指针指向前驱节点
            node.left = pre;
            // 修改当前节点的左指针的类型,指向前驱节点
            node.leftType = 1;
        }
        // 处理后继节点
        if (pre != null && pre.right == null) {
            // 让前驱节点的右指针指向当前节点
            pre.right = node;
            pre.rightType = 1;
        }
        // 每处理一个节点后，让当前节点是下一个节点的前驱节点
        pre = node;

    }

    // ========= 线索化后，各个节点指向有变化，因此原来的遍历方式不能使用 =========

    /**
     * 线索二叉树的遍历
     *
     * @param type 类型
     */
    public void threadedList(TraverseType type) {
        switch (type) {
            case PREORDER:
                preThreadedList();
                break;
            case INORDER:
                inThreadedList();
                break;
            case POSTORDER:
                postThreadedList();
        }
    }


    /**
     * 中序遍历线索化二叉树
     */
    private void inThreadedList() {
        // 定义一个变量，存储当前遍历的节点，从 root 开始
        Node<T> node = root;
        while (node != null) {
            // 循环的找到 leftType == 1 的节点
            // 后面随着遍历而变化,因为当 leftType==1 时，说明该节点是按照线索化处理后的有效节点
            while (node.leftType == 0) {
                node = node.left;
            }
            // 输出这个 leftType == 1 的节点
            System.out.println(node);
            // 如果当前节点的右指针指向的是后继节点,就一直输出
            while (node.rightType == 1) {
                // 获取到当前节点的后继节点
                node = node.right;
                System.out.println(node);
            }
            // 替换这个遍历的节点
            node = node.right;
        }
    }

    /**
     * 前序遍历线索化二叉树
     */
    private void preThreadedList() {
        // 定义一个变量，存储当前遍历的节点，从 root 开始
        Node<T> node = root;
        while (node != null) {
            while (node.leftType == 0) {
                System.out.println(node);
                node = node.left;
            }
            while (node.leftType == 1 && node.right != null) {
                System.out.println(node);
                node = node.right;
            }
            // 说明到最后一个节点
            if (node.right == null) {
                System.out.println(node);
                break;
            }
        }
    }

    /**
     * 后序遍历线索化二叉树
     */
    private void postThreadedList() {
        Node<T> node = root;
        // 找到后序遍历的起始节点
        while (node != null && node.leftType != 1) {
            node = node.left;
        }
        while (node != null) {
            // 如果右节点是线索
            if (node.rightType == 1) {
                System.out.println(node);
                pre = node;
                node = node.right;
            } else {
                // 如果上个处理的节点是当前节点的右节点
                if (node.right == pre) {
                    System.out.println(node);
                    if (node == root) {
                        return;
                    }
                    pre = node;
                    node = node.parent;
                }
                // 如果从左节点的进入则找到有子树的最左节点
                else {
                    node = node.right;
                    while (node != null && node.leftType == 0) {
                        node = node.left;
                    }
                }
            }
        }
    }
}
